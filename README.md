# Blinkit Groceries Sales Analysis
Using Power BI and Excel

📌 Project Objective
This mini project aims to perform real-world retail data analysis using:
     Microsoft Excel for data cleaning and transformation.
     Power BI for interactive dashboard creation and visualization.
The objective is to analyze sales performance, product trends, and outlet efficiency to generate actionable business insights.

📌 Project Overview
This project presents an interactive Blinkit Sales Dashboard built using Power BI to analyze sales performance across products, outlet types, outlet sizes, and establishment years.The dashboard provides actionable insights into revenue trends, product performance, and operational efficiency, helping stakeholders make data-driven decisions. The raw data is cleaned in MS Excel and then the data is import from Excel to Power BI Desktop.

📊 Dataset Source
Dataset: Blinkit Grocery Sales Data
The dataset represents sales transactions across different outlet types, product categories, and locations.

📂 Dataset Description
The dataset includes the following key attributes:
Item Type (Fruits, Snacks, Dairy, etc.)
Outlet Type (Grocery Store, Supermarket Type1/2/3)
Outlet Size (Small, Medium, High)
Outlet Location Type
Outlet Establishment Year
Sales Amount
Item Visibility
Fat Content (Low Fat / Regular)
Customer Rating
